Device List
===========

The method devicelist returns the list of devices owned by the user, and their modules.


Resource URL
^^^^^^^^^^^^

http://localhost/api/ws/devicelist


Medhod
^^^^^^

GET or POST

Required
^^^^^^^^

uuid


Parameters
^^^^^^^^^^

::

 UUID   Your UUID 

Example request
^^^^^^^^^^^^^^^

::

 GET /api/ws/devicelist?uuid=[YourUUID]
